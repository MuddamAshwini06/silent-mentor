# Silent Mentor — Experience Sharing Platform

Silent Mentor is a full-stack web platform where professionals share real career, skills, and academic experiences, and students learn from them.

##  Features

- User Signup & Login (JWT auth)
- Category based experiences:
  - Skills
  - Career
  - Academics
- Post / Edit / Delete own experiences
- View others’ experiences
- Resource sharing per post
- Clean UI with category themes
- Protected routes

##  Tech Stack

Frontend:
- React
- CSS

Backend:
- Node.js
- Express
- MongoDB
- JWT Authentication

##  Structure

client → React frontend  
server → Express backend
